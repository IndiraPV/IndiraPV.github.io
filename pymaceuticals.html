# Use functions to label the percentages of changes

def autolabel(axis):
    rects = axis.patches
    # Get y-axis height to calculate label position from.
    (y_bottom, y_top) = axis.get_ylim()
    y_height = y_top - y_bottom

    for rect in rects:
        height = rect.get_height()

        # Fraction of axis height taken up by this rectangle
        p_height = (height / y_height)
        
        label = '{:.0f}%'.format(height)

        # If we can fit the label above the column, do that;
        # otherwise, put it inside the column.
        if p_height > 0: # arbitrary; 95% looked good to me.
            label_position = height - (y_height * 0.09)
        else:
            label_position = height + (y_height * 0.01)
            
        axis.text(rect.get_x() + rect.get_width()/2., label_position,
                label,
                ha='center', va='bottom',color='white',fontsize=15)

3 Observations are :¶

1. Capomulin looks promising as a cure for the treatment of squamous cell carcinoma (SCC)¶

2. Infubinol was able to control the metastatic spread but given the low survival rate in the process leads me to conclude that it killed the good cells too.¶

3. Ketapril was ineffective in controlling the tumor volume or metastatic spread but had the second highest survival rate. So, ineffective drug with minimal or no side effect.¶

4. Placebo saw the most metastatic growth so the most ineffective of the four.¶
